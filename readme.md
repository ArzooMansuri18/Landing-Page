<h1>Landing Page Prototype</h1>

<p>This project is a clean and modern <b>Landing Page Prototype</b> created using <b>HTML, CSS and JavaScript.</b> It focuses on layout, spacing, typography, and visual hierarchy - exactly what real UI landing pages follow.</p>

<h2>Features</h2>

<ul>
    <li>Hero section with heading, subtext & call-to-action button</li>
    <li>Responsive layout (works on mobile, tablet, desktop)</li>
    <li>Smooth hover effects & button animations</li>
    <li>Clean typography and spacing</li>
    <li>Simple navigation bar (optional)</li>
    <li>Image/illustration section for visual appeal</li>
</ul>

<h2>Technologies Used</h2>

<ul>
    <li>HTML -> Structure of hero, features, navbar, footer</li>
    <li>CSS -> Layout, modern styling, responsiveness</li>
    <li>JavaScript -> For small interactions (scroll, button actions)</li>
</ul>

<h2>How It Works</h2>

<ol>
    <li>Page is divided into important sections:
    <ul>
    <li>Header/Navbar</li>
    <li>Hero Section</li>
    <li>Features Section</li>
    <li>Call-to-Action</li>
    <li>Footer</li>
    </ul>
    </li>
    <li>CSS uses Flexbox/Grid to align elements neatly</li>
    <li>Colors, shadows, and spacing create a premium look</li>
    <li>JavaScript can be used for: 
    <ul>
    <li>Smooth scroll</li>
    <li>Menu toggle (foe mobile)</li>
    <li>Button interactions</li>
    </ul>
    </li>
</ol>

<h2>Future Improvements</h2>
